# ⭐⭐⭐⭐ 告别注意力塌陷：QG-MIL架构解析

**日期**: 2026-06-24

论文 : QG-MIL: A Gated Transformer Aggregator for Domain-Agnostic Multiple Instance Learning in Medical Imaging链接 : https://arxiv.org/abs/2606.20027在医疗影像的弱监督学习中，基于注意力的多实例学习（Multiple Instance Learning, MIL）长期被一个顽疾困扰：注意力塌陷（Attention Concentration）。模型往往把权重全押在少数几个“显眼”的切片上，导致预测过度自信且极不稳定。这篇论文提出的 QG-MIL 提供了一个优雅的解法：不靠复杂的辅助损失或两阶段训练，仅通过重构 Transformer 聚合器的内部结构，就实现了注意力的均匀分布和性能的显著提升。
### 痛点：为什么现有方案不够好？
传统的注意力 MIL 方法（如 ABMIL）容易产生“注意力黑洞”，即权重坍缩到极少数实例上。为了解决这个问题，现有的主流做法是在训练层面动刀：引入注意力掩码、自监督预训练或者师生蒸馏。
这些方法虽然有效，但代价高昂。它们往往引入了额外的训练阶段或辅助损失函数，极大地增加了工程落地的复杂度。对于需要快速迭代和稳定部署的临床管线来说，这种“打补丁”式的正则化方案并不性感。
### 核心洞察：向大模型架构取经QG-MIL 的核心直觉非常清晰：既然注意力塌陷在大规模语言模型中也有类似表现（信息坍缩），那么 LLM 中用于缓解这一问题的架构设计——门控（Gating）和归一化（Normalization），同样适用于医疗影像的 MIL 场景。
作者没有发明全新的算法，而是将四个成熟的组件进行了协同组合：
- RMSNorm 预归一化：替代传统的 LayerNorm，提升训练稳定性。
- Per-head QK 归一化：在计算注意力分数前对 Query 和 Key 进行归一化，防止梯度爆炸或消失。
- 细粒度注意力输出门控：这是最关键的设计。在注意力输出后、投影前，引入一个可学习的线性投影和 Sigmoid 激活函数作为“阀门”，动态调节每个 head 的输出强度。
- SwiGLU 风格的 FFN：使用更高效的前馈网络结构。
这种设计的妙处在于“结构性抑制”。它不是在训练完成后强行拉平注意力，而是通过架构本身让模型“难以”产生极端的注意力峰值。这意味着你可以直接把它作为 Drop-in Replacement（即插即用模块）替换掉现有的 ABMIL 或 TransMIL，无需修改任何训练流程。
### 实验数据：全面碾压基线作者在六个基准数据集上进行了测试，涵盖了全切片病理学（WSI）和细胞级血液学两个截然不同的尺度。结果令人印象深刻：
1. 整体性能提升QG-MIL 变体在所有六个基准测试中均优于现有基线，平均宏观 F1 分数提升了 +6.1 个点。
2. 病理诊断任务（Table 1）
- 乳腺癌（Breast）：使用 CONCHv1.5 编码器时，QG-MIL Deep 达到了 98.97% 的 F1 分数，远超基线平均水平的 85.94%。
- 肺癌（Lung）：在数据量较少的 LungHist700 数据集上，QG-MIL no Gate 变体达到了 93.34% 的 F1 分数，显著优于之前研究中报告的 81.6%。
3. 预后预测任务（Table 2）
前列腺癌预后是一个极具挑战性的任务（从良性预测未来癌症风险）。QG-MIL Layernorm 变体取得了最高的平均宏观 F1 57.55% ，而基线方法的最高分仅为 Transformer 的 56.97%。
4. 血液学诊断（Table 3）
在细胞级别的分类任务中，QG-MIL 同样表现出色：
- AML 亚型分类：达到 86.02% 的 F1 分数，超越专用 MIL 方法 5%。
- cAItomorph 数据集：QG-MIL Deep 取得了 67.97% 的加权 F1 分数，比之前评估的 8 层 Transformer 模型高出 3.7%。
### 注意力分布：从“尖峰”到“平滑”
除了准确率，QG-MIL 在可解释性上也更胜一筹。论文通过基尼系数（Gini Index）和熵来量化注意力的均匀程度：
- ABMIL：Gini 指数高达 0.62，前 10% 的实例占据了总注意力的 54%，呈现出典型的“尖峰”分布。
- QG-MIL：Gini 指数降至 0.16，前 10% 实例的注意力占比稳定在 15% 左右。
这意味着 QG-MIL 能够更均匀地关注图像中的不同区域，避免了因局部噪声导致的误判，生成的热力图也更加平滑、符合临床直觉。
### 工程启示与局限⚠️ 数据规模决定架构复杂度这是一个反直觉的发现：在患者数量较少的小数据集上，完整的门控机制反而可能引入方差，导致泛化能力下降。论文建议在小样本场景下使用简化版（如 QG-MIL Light 或 no Gate），而在中大规模数据集上，深层结构（QG-MIL Deep）能带来显著收益。
计算成本可控尽管引入了门控和 SwiGLU，QG-MIL 的推理开销并不高昂。对于常见的 512 和 1024 个实例的 Bag Size，其计算量分别约为 7 GFLOPs 和 18 GFLOPs，低于 RRT 和 TransMIL。
总结QG-MIL 证明了通过精心设计的架构组件组合，可以在不增加训练复杂度的前提下，解决 MIL 中的注意力塌陷问题。对于正在构建医疗影像 AI 管线的工程师来说，这是一个值得尝试的“即插即用”升级方案，尤其是在数据量充足的情况下，它能提供更稳定、更可解释的诊断辅助能力。

挑刺:
- 博文声称“QG-MIL 变体在所有六个基准测试中均优于现有基线”，但原文明确写的是“The best-performing QG-MIL variants outperform leading baselines on all six benchmarks”。例如在Prostate预后任务中，QG-MIL no Gate平均macro F1为53.03%，低于Transformer的56.97%和DSMIL的56.14%（原文Table 2）。该表述将“最佳变体”泛化为“所有变体”，可能造成读者误解。
