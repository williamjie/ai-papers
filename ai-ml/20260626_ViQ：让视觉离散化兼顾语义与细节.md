论文 : ViQ: Text-Aligned Visual Quantized Representations at Any Resolution链接 : https://arxiv.org/abs/2606.27313在 MLLM 时代，我们常陷入一个尴尬的权衡：连续视觉特征（Continuous Features）语义强但计算重，离散视觉 Token（Discrete Tokens）轻量却丢失细节。ViQ 的出现打破了这一僵局，它证明了离散表示不仅能媲美连续特征的语义理解能力，还能实现原生分辨率支持，并在训练效率上带来显著加速。
### 痛点：离散化的“失忆”困境现有的离散视觉编码器面临一个核心矛盾： 重建导向的模型缺乏语义，而语义强的模型在量化时丢失细节 。
传统方法如 CLIP 或 SigLIP 依赖连续高维特征，虽然与 LLM 交互效果好，但存在模态间的表示鸿沟，且硬件负担重。早期的离散化尝试（如 QLIP、UniTok）试图将图像压缩为有限 Token 集合，却因压缩比过高导致严重信息损失。结果就是：要么看得清但看不懂，要么看得懂但看不清。ViQ 的核心洞察在于，必须通过 两阶段解耦学习 来平衡这一矛盾：先对齐语义，再约束空间进行量化。
### 方法拆解：两步走的“降维打击”
ViQ 的设计并非简单的堆叠模块，而是对特征空间的精细雕刻。其核心流程分为两个阶段：
-文本对齐预训练（Text-Aligned Pre-training）
原生分辨率支持：借鉴 NaViT 思路，替换位置编码层以支持动态缩放。这解决了固定分辨率裁剪导致的细节丢失问题。
- 自蒸馏正则化：为了防止模型在适应多模态任务时“遗忘”基础视觉知识，ViQ 使用固定分辨率的教师模型监督学生模型的 Class Token。公式 Ldistill=1−cos⁡(zsstudent,zsteacher)\mathcal{L}_{\text{distill}} = 1 - \cos(\mathbf{z}_s^{\text{student}}, \mathbf{z}_s^{\text{teacher}})​=1−cos(zsstudent​,zsteacher​) 确保了任意分辨率下的高层语义一致性。
-视觉量化学习（Visual Quantization Learning）
邻近表示学习（Proximal Representation）：这是 ViQ 的杀手锏。直接量化高维特征会导致精度崩塌，ViQ 引入瓶颈层压缩维度，并使用 L∞L_\infty​-norm 将特征投影到超立方体表面（∥f1∥∞=1\| f_1 \|_\infty = 1​∥∞​=1）。这种约束极大地降低了特征空间复杂度，减少了量化锚点间的距离误差。
- 多头有限标量量化（Multi-Head FSQ）：采用无需额外优化的 FSQ 算法，并结合多头注意力机制将每个 Patch 扩展为 2×22 \times 22 的视觉代码。这不仅增加了表示容量，还通过独立处理保持了 Token 的独立性。
- 2D RoPE 注入：在量化前插入 2D 旋转位置编码，显式保留空间分辨率信息，确保不同尺寸图像的空间关系不被破坏。
### 关键结果：性能与效率的双赢ViQ 在多个基准测试中展现了惊人的竞争力，尤其是在保持离散特性的同时，缩小了与连续模型的差距。
多模态理解能力（Table 1 & 2）
以 Qwen2.5-1.5B 为底座，ViQ 在 9 个基准的平均得分达到 57.2 ，超越了参数更多或结构更复杂的连续编码器：
- InternViT-2.5 (0.3B): 56.5- SigLIP2-g (1.1B): 53.1- QLIP-B (0.3B, 离散): 29.7- ViQ (1.3B, 离散): 57.2更令人印象深刻的是，在 Qwen2.5-7B 底座下，ViQ 平均分达到 63.9 ，略微超越 InternViT-2.5-6B (63.8)。这意味着一个 1.3B 参数的离散编码器，配合小 LLM，能达到与 6B 连续编码器相当的效果。
训练效率提升（Fig 3）
由于 ViQ 可以离线提取离散代码，训练时无需加载原始图像，仅加载预计算的 Codebook 索引并投影到 LLM 空间。这带来了显著的加速：
- Qwen2.5-0.5B: 前向传播时间加速 70% (4k序列) / 78% (16k序列)。
- Qwen2.5-7B: 前向传播时间加速 46% (4k序列) / 65% (16k序列)。
- 整体迭代步骤（Step Time）也有 20%-40% 的提升。
重建质量（Table 3）
ViQ 并未牺牲底层细节。在 ImageNet-1K 验证集上，ViQ 的 rFID 为 0.62 ，优于 UniTok (0.37 但理解能力极弱) 和 QLIP-B (3.21)。PSNR 达到 22.73，证明了其在离散化后仍保留了足够的重建信息。
### 工程启示对于从事 MLLM 微调或部署的工程师，ViQ 提供了极具价值的参考：
- 存储成本骤降：ViQ 将图像转换为 16-bit 整数序列，存储空间仅为原始图像的 1/96。这对于需要长期存储多模态数据的 Agent 系统至关重要。
- 训练管线优化：离线提取 Codebook 的策略解耦了视觉编码与 LLM 训练。这不仅加速了训练，还简化了数据预处理流程，避免了每次迭代都运行昂贵的 ViT 前向传播。
- 原生分辨率的价值：ViQ 的实验表明，支持任意分辨率（Native Resolution）能显著提升 OCR 和文档理解任务的表现。在工程落地中，避免强制裁剪或缩放图像是提升细粒度任务性能的关键。
### 局限与展望尽管 ViQ 表现优异，但在极端细节任务上仍有局限。论文指出，在 OCRBench 等高频细节密集型任务上，ViQ 仍略逊于部分连续编码器。这归因于离散 Token 化固有的信息损失。未来可通过多尺度量化或残差连接进一步缩小这一差距。
总结 ：ViQ 证明了“离散化”不再是性能的妥协，而是一种高效的统一表示范式。对于追求训练效率和存储优化的团队，ViQ 是一个值得深入研究的 SOTA 方案。
