论文 : Geometric Action Model for Robot Policy Learning链接 : https://arxiv.org/abs/2606.17046当前主流机器人策略模型（如 VLA 或 WAM）大多在 2D 图像空间“硬猜”深度和遮挡，导致面对相机视角变化时鲁棒性极差。这篇论文提出的 GAM（Geometric Action Model）直接复用几何基础模型（GFM），让机器人在显式 3D 潜空间中思考，不仅推理速度提升了 55 倍，还在相机扰动测试中取得了显著优势。
### 为什么现有方案不够用？
现有的视觉-语言-动作模型（VLA）虽然语义理解强，但本质上是 2D 的。它们依赖单目线索隐式推断深度，一旦相机位置改变或光照变化，泛化能力就断崖式下跌。
另一种思路是视频世界模型（WAM），通过预测未来帧来辅助决策。但这同样受限于 2D 像素空间，无法显式解决尺度、遮挡等几何歧义。
虽然已有工作尝试引入 3D 几何信息，但大多只是把 GFM 当作静态特征提取器（Feature Extractor），并未真正利用其多层结构进行时间序列预测。GAM 的核心洞察在于： 既然 GFM 已经理解了 3D 结构，为什么不直接在这个潜空间里做时序预测？
### GAM 的核心设计：拆分与注入GAM 没有从头训练一个大模型，而是巧妙地“手术”了一个预训练的几何基础模型（如 DA3-Giant）。
- 切分骨干网：将 GFM 在中间层（Layer 12）切开。浅层作为观察编码器，提取当前帧的几何潜特征；深层保留作为解码器。
- 插入因果预测器：在切分点插入一个因果 Transformer。它接收当前的几何特征、语言指令、本体感知和历史动作，直接预测未来的几何潜 Token。
- 联合解码：预测出的未来几何 Token 与动作 Token 一起送入深层 GFM。利用 GFM 原有的深度头（Depth Head）和解码能力，同时输出未来几何结构和当前动作 Chunk。
这种设计的精妙之处在于 共享表征空间 。动作不再是孤立回归出来的，而是与未来的 3D 几何变化紧密耦合。模型在预测“下一步怎么走”时，必须同时理解“世界会变成什么样”。
### 实验结果：又快又准GAM 的性能提升不仅体现在准确率，更体现在极致的效率上。以下是关键数据对比：

### 工程启示对于正在落地机器人策略的工程师，这篇论文提供了几个重要参考：
- 不要重复造轮子：直接复用成熟的几何基础模型（GFM）作为骨干，比从头训练 VLA 更高效且具备更强的空间理解力。
- 潜空间预测优于像素预测：在 GFM 的中间层进行时序预测，比在像素空间或高维语义空间预测更稳定，且计算成本更低。
- 切分层选择至关重要：论文消融实验显示，切分层 Ls=12L_s=12​=12 是最佳平衡点。太浅特征不足，太深则破坏了 GFM 原有的几何解码逻辑。
### 局限与展望GAM 目前依赖冻结的 T5 文本编码器，语言推理能力有限。未来若结合更大的 LLM 或外部推理模块，可能会进一步提升复杂指令遵循能力。此外，虽然它在模拟和少量真实任务中表现优异，但在大规模多样化真实场景中的泛化边界仍需更多验证。
