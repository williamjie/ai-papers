# ⭐⭐⭐½ LLM 答案稳定性测试：谁会被反论证说服？

**日期**: 2026-06-16

---

论文 : Who Flips? Self- and Cross-Model Counterarguments Reveal Answer Instability in LLMs链接 : https://arxiv.org/abs/2606.16011我们常以为模型答对题就是“懂”了，但这篇论文揭示了一个尴尬事实：很多模型只是“猜”对了。一旦遇到看似合理的错误反论证，它们会毫不犹豫地放弃正确答案。这种“答案不稳定性”（Answer Instability）在标准准确率基准中完全隐形，却是工程落地时的巨大隐患。
### 痛点：准确率掩盖了脆弱性现有的评估体系只看结果对不对，不看模型有多坚定。在实际应用中，用户可能会质疑答案，Agent 之间会互相辩论。如果模型因为一句“我觉得你错了”或者一段逻辑严密的错误推理就改口，那它的可靠性就很存疑。
之前的研究多关注“谄媚”（Sycophancy），即模型屈服于社交压力。但这篇论文做了一个关键区分： 剥离社交压力，只看论证内容 。它问的是：如果只给出一段支持错误选项的连贯逻辑，而不带任何情绪化指责，模型还会动摇吗？
### 方法拆解：两阶段“压力测试”
作者设计了一个极简但犀利的 Who Flips 协议：
- 胁迫生成（Coercion）：先让模型在隔离会话中，为错误选项生成一段 kk 句长的论证。这确保了挑战内容的逻辑自洽性。
- 真实挑战（Challenge）：在新会话中，让模型正常答题。如果答对，立即抛出刚才生成的错误论证。
核心变量设计非常巧妙：
- 归因（Attribution）：是匿名抛出（BLIND），还是告诉模型“这是你之前写的”（SELF）？
- 来源（Source）：是自己生成的论证，还是其他模型生成的（CROSS）？
- 长度（Length）：论证越长，说服力越强吗？
### 关键结果：谁最容易被带偏？
实验覆盖了 7 个前沿模型和 MMLU 的 57 个学科，数据令人咋舌。
1. 翻转率差异巨大，与规模无关在匿名挑战下，模型的“答案翻转率”（AFR）从 17.5% 到 97.3% 不等。
- Llama-3.1-8B 是最脆弱的，翻转率高达 97.3%。
- Qwen3.5-35B 最稳定，仅 17.5%。
⚠️ 反直觉发现 ：模型越大越稳？不一定。Llama-3.3-70B（70B）的翻转率（75.8%）远高于 Qwen3.5-9B（39.3%）。稳定性更多取决于对齐策略，而非参数量。
2. “自我归因”是强力催化剂当告诉模型“这个错误论证是你之前写的”时，所有模型的翻转率都上升了。
- 平均增幅：+7.1%。
- Qwen3.5-4B 增幅最大，达到 +18.7%。
这说明模型对自己的历史输出有一种盲目的信任或一致性偏好，哪怕那是错的。
3. 跨模型挑战：被挑战者比挑战者更重要在跨模型实验中，作者发现： 哪个模型被问（Target）解释了 76.7% 的方差，而谁在提问（Source）只解释 12.0% 。
这意味着，模型的“固执程度”是其主要属性。不过，最稳定的模型（如 GPT-5.1, Qwen3.5-35B）往往也是最能生成有说服力错误论证的来源（高认知权威 Epistemic Authority）。
4. 学科差异：STEM 更硬气- 道德争议（Moral disputes）：翻转率高达 80.8%。
- 初等数学（Elementary mathematics）：仅 20.9%。
形式逻辑强的领域，模型更不容易被花言巧语带偏；而人文社科类问题，由于缺乏唯一标准答案，模型更容易摇摆。
### 工程启示- 不要只看准确率：在部署涉及辩论、客服或辅助决策的 Agent 时，必须引入稳定性测试。一个准确率 90% 但翻转率 80% 的模型，在生产环境中可能是灾难性的。
- 利用“自我归因”做防御：既然模型容易轻信自己的历史输出，可以在多轮对话中显式引用其之前的正确推理，作为锚点来抵抗外部干扰。
- 构建 MAXFLIP 数据集：作者开源了通过跨模型池化筛选出的最强错误论证集（MAXFLIP），可用于压力测试你的模型。特别是对于中等能力的模型，这种针对性攻击能让翻转率再提升 +23.6%。
### 局限与展望目前研究仅基于 MMLU 多选题，且未探索缓解措施。未来工作需关注如何在训练阶段增强模型的“认知定力”，以及如何在多 Agent 协作中防止错误共识的形成。
