论文 : MBench: A Comprehensive Benchmark on Memory Capability for Video World Models链接 : https://arxiv.org/abs/2606.00793现在的视频生成模型，画质确实卷到了极致。但如果你让它模拟一个真实的物理世界，问题就来了：镜头转个身，桌上的杯子是不是还在？爆炸之后，废墟是不是还在？
大多数现有评测只关心“像不像”，却忽略了世界模型最核心的能力—— 长期记忆（Long-term Memory） 。清华与腾讯联合提出的 MBench，就是为了解决这个痛点，给视频世界模型做了一次彻底的“记忆力体检”。
### 为什么现有的评测不够用？
目前的 Video Generation Benchmarks（如 VBench, T2V-CompBench）主要关注视觉质量、运动连贯性和文本对齐。
这导致了一个严重的误导：一个模型可能生成了每一帧都很逼真的视频，但在长时序下完全“失忆”。比如镜头移开再回来，原本红色的车变成了蓝色；或者爆炸发生后，镜头穿过烟尘，发现街道完好无损。
MBench 的核心 Insight 在于： 世界模型的本质不是生成图片，而是维护状态。 因此，它将记忆力拆解为三个层级：
- 实体一致性（Entity Consistency）：物体和人的身份、属性是否保持不变？
- 环境一致性（Environment Consistency）：空间布局、光照风格是否稳定？
- 因果一致性（Causal Consistency）：物理规律和交互后果是否逻辑自洽？
### 方法拆解：如何量化“记忆力”？
MBench 的设计非常工程化，它没有依赖模糊的主观打分，而是引入了 触发条件评分（Trigger-Conditioned Scoring） 。
⚠️ 关键设计直觉 ：很多模型为了刷高分，会生成静止或保守的内容来避免错误。MBench 强制模型执行“离焦-回归”等复杂动作，只有成功触发了挑战，才计算一致性得分。最终分数是触发覆盖率与记忆可靠性的调和平均数（Harmonic Mean），防止“躺平”刷分。
具体指标设计极具巧思：
- 几何一致性：利用 SAM 2 提取物体掩码，通过图像扭曲（Warping）对齐前后帧，计算 SSIM。这能精准捕捉物体在遮挡后回归时的形变。
- 纹理一致性：使用 DINOv2 提取特征，计算当前帧与全局轨迹质心的余弦相似度。这比相邻帧对比更能反映长期漂移。
- 因果演化：引入 VLM 作为裁判，评估“离屏期间”的物理变化是否正确发生（如爆炸后的废墟）。
### 实验结果：主流模型普遍“健忘”
MBench 评测了包括 Memflow, Self Forcing, SkyReels V2 在内的 12 个 SOTA 模型。数据揭示了令人担忧的现状： 没有哪个模型在长时序记忆上表现完美。

- 人类身份一致性极差：SkyReels V2 在物体几何上得分高达 70.03，但在人类身份识别上仅为 24.57。这意味着模型能记住桌子的形状，却记不住人的脸。
- 因果推理是短板：Self Forcing 的空间一致性尚可（67.44），但因果演化得分仅 50.19。说明它能维持空间结构，但不懂物理逻辑。
- 没有全能选手：Memflow 在物体几何上表现最好（61.72），但在人类身份和光照一致性上明显落后。
### 工程启示对于正在落地视频世界模型的工程师，MBench 提供了明确的优化方向：
- 不要只调参画质：如果目标是 Agent 或仿真环境，必须引入显式的记忆模块（如外部记忆库 External Memory 或参数化记忆 Parametric Memory）。
- 重视“离屏”测试：在评估模型时，务必设计镜头移开再回来的场景。这是检验世界模型是否真正理解“物体恒存性”的金标准。
- 因果一致性需专项优化：目前的扩散模型大多基于统计规律，缺乏物理引擎般的因果约束。结合 VLM 进行因果校验，可能是提升逻辑一致性的有效路径。
### 局限与展望MBench 目前主要依赖真实视频作为 Ground Truth，对于完全虚构的场景（如科幻电影）覆盖有限。此外，VLM 评估器本身可能存在幻觉，未来需要更客观的自动化指标。
但无论如何，MBench 撕开了视频生成“视觉欺骗”的面纱，提醒我们： 真正的世界模型，必须是一个守信用的记忆者。
