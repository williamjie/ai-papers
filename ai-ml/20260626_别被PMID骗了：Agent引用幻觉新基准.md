# ⭐⭐ 别被 PMID 骗了：Agent 引用幻觉新基准

**日期**: 2026-06-26

---

论文 : OpenBioRQ: Unsolved Biomedical Research Questions for Agents链接 : https://arxiv.org/abs/2606.21959现在的医疗 Agent 越来越像“学术骗子”——它们给出的 PMID 是真的，文章也是真的，但内容完全对不上号。
这篇论文揭示了一个被现有基准严重低估的失败模式： 引用存在性不等于内容支持 。对于构建严肃科研辅助工具的工程团队来说，这是一个必须正视的红线问题。
### 痛点：有答案的测试掩盖了“伪证”
现有的医疗 QA 基准（如 MedQA、BioMysteryBench）都有一个共同假设：问题有标准答案。
在这种设定下，Agent 只需要背诵或检索出那个“正确来源”，就能通过测试。这导致两个盲区：
- 无法检测错误归因：如果 Agent 引用了一篇真实但无关的论文，只要它碰巧提到了正确答案，传统评测就认为它是对的。
- 缺乏对“未知”的敬畏：面对无解的开放性问题，Agent 往往倾向于编造答案，而不是承认不确定性。
### 核心 Insight：两层审计与“冻结清单”
作者提出了 OpenBioRQ，一个包含 12,553 个未解决生物医学研究问题的基准。其设计精髓在于 剥离了标准答案的束缚 ，转而考察 Agent 的证据链完整性。
1. 双层引用审计（Two-Level Citation Audit）
这是本文最反直觉的发现来源。作者将引用质量拆解为：
- L1 存在性：PMID 是否指向真实论文？
- L2 内容支持：该论文是否真正支撑了 Agent 的论点？
⚠️ 关键发现 ：在生物医学领域，Agent 极少捏造不存在的 PMID（仅 0.7%），但高达 15.9% 的真实引用是“张冠李戴”（Wrong-paper）。
相比之下，数学领域的 ResearchMath 基准显示近 54% 的引用是彻底捏造的。这种差异意味着：在医疗场景下， “真实的错误”比“虚假的正确”更具欺骗性 。
2. 冻结清单评分（Frozen Checklist Scoring）
没有标准答案，怎么打分？作者不使用模糊的 LLM-as-a-Judge 维度评分，而是为每个问题生成一份 冻结的检查清单 （Frozen Checklist）。
- Must Mention：必须提及的关键机制。
- Must Acknowledge：必须承认的不确定性或知识空白。
- Must Ground：必须提供真实证据支持的论点。
- Must Avoid：禁止出现的断言（如声称未解之谜已有定论）。
这种设计将主观评分转化为可审计的二元判定，将评委间的一致性（Spearman 相关系数）从 0.35 提升至 0.82 。
### 实验结果：前沿模型依然“翻车”
在由三个开源参考模型共同判定的“核心难题集”（Core Set, n=657）上，结果令人深思：
模型 核心集解决率 (Solve Rate) 主要失败模式 GPT-4o ~29% 引用错误、过度自信 Opus-4.7 ~38% 工具使用效率低 Gemini-3-Pro ~60% 最佳表现，仍有 40% 未解几个值得工程关注的现象：
- Agent 崩溃（Agentic Collapse）：在最难的问题上，部分模型会停止调用工具，直接依靠参数记忆胡言乱语。对于最容易崩溃的模型，禁用工具后其得分几乎没有变化——说明它在最难的地方根本没在用工具。
- 工具并非万能：在核心集上，GPT-4o 使用工具的解决率为 59.6%，不使用工具为 55.6%。置信区间重叠，表明在当前难度下，工具调用并未带来统计显著的收益。
- 解码敏感性：温度（Temperature）设置对难度判定影响巨大。T=0.3 时几乎无人能解，T=0 时解决率显著上升。这提示我们在评估 Agent 能力时，必须固定解码参数。
### 工程启示- 重构引用验证逻辑：如果你的应用允许用户点击 PMID 跳转，请务必增加 L2 检查。不要只校验链接有效性，要用轻量级模型或向量检索比对摘要与论点的相关性。
- 鼓励“承认无知”：在 Prompt 工程中，显式奖励“Must Acknowledge”行为。对于未解决问题，高质量的回答是明确指出知识边界，而非强行给出一个看似合理的结论。
- 警惕工具幻觉：监控 Agent 的工具调用轨迹。如果模型在复杂查询中突然停止调用工具并直接输出长文本，这往往是性能崩溃的信号，应触发重试或人工介入。
### 局限与展望OpenBioRQ 目前仍依赖 LLM 进行引用支持度判断（尽管一致性已很高），尚未完全替代人类专家审计。此外，基准侧重于文献综合，不涉及临床决策支持，切勿用于患者护理场景。
未来，随着“未解决问题”库的扩大，我们有望看到 Agent 在 证据溯源 和 不确定性量化 上的真正突破。在此之前，小心那些看着很真、实则无关的 PMID。
